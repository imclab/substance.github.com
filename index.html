Substance for Developers

Substance provides an extensive, customizable content creation and annotation framework that enables user-centric interfaces and automatic data processing via your web-browser. Thanks to its modular nature it creates an ideal environment for many integration scenarios. Please have a look at our Manual and check our Blog for regular updates.

Substance Surface

Substance Surface is a visual editing component, that you can use to power any kind of content input. By default it supports all features Markdown offers (headings, paragraphs, lists, images, links, ...) but it can be extended easily.

Substance IO

A minimal static publishing system based on Markdown. Read the introduction.

Substance Document

Substance Document allows manipulating structured digital documents and provides an open exchange format based on JSON. It ensures consistency, separates content from presentation and offers an easy to use API.

Substance Converter

The Substance Converter allows working with existing formats, such as Markdown, LaTeX, MediaWiki. Take the offical Substance Tutorial and learn how to generate your first Substance document.
